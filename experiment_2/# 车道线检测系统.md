# 车道线检测系统

## 项目概述
本项目使用霍夫变换实现车道线检测功能，适用于道路图像中的车道线识别。

## 实现方法
1. **图像预处理**：灰度转换、高斯模糊、Canny边缘检测
2. **感兴趣区域提取**：仅保留图像中可能包含车道线的区域
3. **霍夫变换**：检测图像中的直线
4. **车道线处理**：合并和延伸检测到的直线
5. **结果可视化**：将检测到的车道线绘制到原始图像上

## 依赖环境
- Python 3.x
- OpenCV
- NumPy

## 使用说明
1. 安装依赖：`pip install opencv-python numpy`
2. 下载测试图像：运行 `python download_test_image.py`
3. 执行车道线检测：运行 `python lane_detection.py`

## 结果输出
- 程序会显示检测结果窗口
- 检测结果会保存为 `lane_detection_result.jpg`

## 注意事项
- 用户可以替换 `road_image.jpg` 为自己拍摄的道路图像
- 对于不同的图像，可能需要调整参数以获得最佳检测效果
